SERVER:
Database commands:
    login - gets a userobject and validates it. Sends back a userobject with login-boolean either true or false.
    createUser - gets a userobject, persists it to the database. Sends back response.
    updateUser - gets a userobject, updates it to the database. Sends back response.

Game commands:
    startGame - gets a userobject of the player. Sends the userobject and a reference to the networkcommunication to the gamecontroller.
    stopGame - gets a userobject of the player. Sends it further to the gamecontroller.
